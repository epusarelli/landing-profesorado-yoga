Las tipografías (Newsreader, Lato y Dancing Script) se cargan desde Google Fonts
mediante la etiqueta <link> en el <head> de cada página. No se requieren archivos
locales. Si prefieres alojarlas aquí para funcionamiento 100% offline, descarga las
familias y reemplaza el <link> por reglas @font-face en assets/css/style.css.
